# Contributing to Task Manager

Thank you for your interest in contributing! Since this is a small personal project, the guidelines are kept simple.

## Branch Naming Convention

Please create a branch from `main` using one of the following prefixes:
- `feat/`: For new features
- `fix/`: For bug fixes
- `docs/`: For documentation updates
- `chore/`: For maintenance or minor changes

Example: `feat/add-dark-mode`

## Commit Message Rules

- Use the imperative mood (e.g., "Add feature" not "Added feature" or "Adds feature").
- Keep it to one logical change per commit.
- Use a prefix describing the change type:
  - `feat:` for new features
  - `fix:` for bug fixes
  - `docs:` for documentation
  - `chore:` for routine tasks

Example: `feat: add localStorage support for tasks`

## Local Setup

1. Fork and clone the repository.
2. Open `index.html` directly in your browser.
3. Make your changes in a new branch.
4. Test manually in the browser.
5. Submit a pull request!
