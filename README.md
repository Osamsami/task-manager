# Task Manager

A simple, vanilla HTML/CSS/JavaScript to-do list app.

## Features

- Add task
- Mark complete
- Delete task
- Filter tasks
- Save to localStorage

## Setup Instructions

1. Clone or download this repository.
2. Open `index.html` in your web browser. No servers or build tools required!

## Project Structure

- `index.html`: The main page layout.
- `style.css`: Styling for the application.
- `script.js`: Application logic.

## Manual Testing

1. Open `index.html`.
2. Type a task in the input field and press Enter or click the Add button.
3. Verify the task appears in the list.
4. Click a task to mark it as complete.
5. Click the delete button next to a task to remove it.
6. Use the filter buttons to toggle between All, Active, and Completed tasks.
7. Refresh the page to ensure your tasks are saved locally.
